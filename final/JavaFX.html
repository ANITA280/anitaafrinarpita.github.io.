Java And JavaFX projects with Video Demo

JavaFX Calculator Desktop Application

The JavaFX Calculator Application is a graphical user interface (GUI) program built using JavaFX, a framework for creating cross-platform desktop applications. This calculator application provides a user-friendly interface for performing basic arithmetic operations such as addition, subtraction, multiplication, and division.

JavaFX Canvas Desktop Command Line Application

The Canvas Designer Application is a user-based program that allows users to create and design canvases with various elements such as text, images, circles, and rectangles. Users can create a profile, log in, edit their profile information, create canvases with customizable dimensions, and perform various design actions on the canvas. The application provides features such as zooming, changing the background color, clearing the canvas, exporting the canvas as an image, adding, moving, rotating, resizing, and deleting elements. Users can also customize the properties of individual elements such as text content, font, color, alignment, and border settings. The application ensures the persistence of user profile data and provides a version check feature.

Java BnB Application

The Melbourne Bnb Application is a command-line program built using Java that allows users to search and book properties in Melbourne. The application reads property details from a CSV file, provides various search and filter options, calculates prices based on user inputs, and allows users to make reservations.
